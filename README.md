**Welcome to Sagar Gyalan Magar project** 

**About**

View and Edit  your app on [sagargyalan.com.np](http://Sagargyalan.com.com) 

This project contains everything you need to run your app locally.

**Edit the code in your local development environment**

Any change pushed to the repo will also be reflected in the Sagar Gyalan Magar Builder.

**Prerequisites:** 

1. Clone the repository using the project's Git URL 
2. Navigate to the project directory
3. Install dependencies: `npm install`
4. Create an `.env.local` file and set the right environment variables

```
VITE_sagargyalan.com.np_APP_ID=your_app_id
VITE_sagargyalan. com. np_URL=your_backend_url

e.g.
VITE_sagargyalan. com. np_ID=cbef744a8545c389ef439ea6
VITE_BASE44_APP_BASE_URL=https://my-to-do-list-81bfaad7.sagargyalan.com.np
```

Run the app: `npm run dev`

**Publish your changes**

Open [sagargyalan. com. np](http://sagargyalan.com.np) and click on Publish.
